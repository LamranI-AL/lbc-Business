Développement du Site Centre Laser

**Objectif :** Créer un site web professionnel pour un centre d'épilation laser médicale, inspiré de l'architecture et de l'expérience utilisateur de Maison du Laser, avec une identité visuelle moderne et distinctive.

### Stack Technique :
- **Frontend :** Next.js 14+ avec TypeScript et App Router
- **Styling :** Tailwind CSS avec design system personnalisé
- **Animations :** GSAP pour micro-interactions et animations fluides
- **Backend :** Firebase v9+ (Firestore, Authentication, Storage)
- **Images :** Next.js Image avec optimisation automatique
- **Déploiement :** Vercel avec CI/CD

### Architecture du Site et Fonctionnalités :

- Menu horizontal avec sous-menus déroulants (Services → Visage/Aisselles/Jambes/Maillot)
- Sticky header avec logo et CTA "Prendre RDV" prominents

- Numéro de téléphone visible en permanence

#### 5. **Système de Réservation Avancé**

**Fonctionnalités requises :**
- Sélection du centre (Paris 11, Paris 12, Paris 14, Paris 15, Argenteuil, Bruxelles)
- Choix du service par catégorie
- Calendrier intelligent avec créneaux disponibles
- Gestion des tarifs promotionnels (-50% Summer Body)
- Formulaire client avec informations médicales
- Confirmation automatique par email/SMS

**Structure Firebase pour réservations :**
```javascript
reservation: {
  id: string,
  centreId: string,
  serviceIds: string[],
  clientInfo: {
    nom: string,
    prenom: string,
    email: string,
    telephone: string,
    age: number,
    typesPeau: string
  },
  dateRdv: timestamp,
  heureRdv: string,
  prixTotal: number,
  statut: 'confirmé' | 'en_attente' | 'annulé',
  consultationOfferte: boolean
}
```

#### 7. **Composants UI Spécifiques**

**Composants à développer :**
- `PricingTable` : Tableau tarifs avec colonnes normales/promos
- `ServiceSelector` : Sélecteur de services par zone corporelle
- `CentreLocator` : Carte avec géolocalisation des centres
- `BookingCalendar` : Calendrier de réservation interactif
- `TreatmentProcess` : Timeline du processus de traitement
- `BeforeAfterGallery` : Galerie avant/après avec filtres
- `FAQ` : Accordéon avec questions fréquentes par service
- `TestimonialCarousel` : Carrousel d'avis clients avec étoiles

#### 8. **Design System**

**Palette de couleurs :**
- Or/Bronze : `#B8860B` (couleur principale)
- Bleu ciel : `#87CEEB` (accents)
- Blanc : `#FFFFFF` (backgrounds)
- Gris foncé : `#2D2D2D` (textes)
- Rouge : `#DC2626` (promotions)

**Typography :**
- Titres : Police élégante (Playfair Display ou similaire)
- Corps de texte : Police lisible (Inter ou similaire)
- Hiérarchie claire avec tailles définies

#### 9. **Animations GSAP Spécifiques**

**Micro-animations :**
- Parallax subtil sur hero sections
- Révélation progressive des tarifs au scroll
- Hover effects sur cartes de services
- Transition fluide entre pages
- Loading animation pour le calendrier

2. **Système de réservation** opérationnel

4. **Documentation technique** complète

**Note Cruciale :** Le site doit reproduire l'expérience utilisateur premium de Maison du Laser tout en créant une identité visuelle unique. L'accent doit être mis sur la conversion (réservations) et la crédibilité médicale.
